# Mariu-cidade

Jogo de plataforma 2D estilo Mario feito em HTML5 Canvas. 
Jogue direto no navegador, no celular ou PC.

## Jogar

**Online:** https://detillimichel-max.github.io/Mariu-cidade/

Ou abra o arquivo `index.html` localmente.

## Controles

- **Celular:** ◀ ▶ para andar, ▲ para pular
- **PC:** Setas ← → para andar, Espaço / ↑ para pular

## Objetivo

Colete moedas 🪙, pise nas tartarugas 🐢 e chegue na bandeira.

## Tecnologias

- HTML5 Canvas
- JavaScript puro
- Sem dependências
